更新项目仓库

🔄

用 git pull 命令一键更新项目

项目有新内容？只需一条命令，就能让你的本地项目和远程仓库保持同步！

1
📂

进入项目文件夹

在终端输入 cd 项目文件夹名，进入你要更新的项目目录。

🖥️ 终端
$ cd 项目文件夹名
2
🔄

更新项目

输入 git pull，就能把最新的项目内容下载到你的电脑！

🖥️ 终端
$ git pull
正在更新项目...
✅ 更新完成！
💡
完整语法： git pull <remote> <branch>
git pull origin main - 从origin远程仓库的main分支拉取
git pull origin master - 从origin远程仓库的master分支拉取
🎉

恭喜！你的项目已经是最新的啦！

记得经常用 git pull 保持项目同步哦！

🎯

真实项目实战演练

跟着我们一起下载并运行一个真实的Python小游戏项目！

1
📥

下载项目

使用Git Clone命令下载猜数字游戏项目

🖥️ 终端命令
$ git clone https://github.com/aicoder2048/guess_num.git
正在克隆到 'guess_num'...
✅ 下载完成！
2
📖

阅读说明书

重要！下载项目后第一件事：看README.md文件了解项目

⚡

为什么要先看README.md？

📋 了解项目是做什么的
🛠️ 学习如何安装和运行
📝 查看使用说明和注意事项
🖥️ 进入项目文件夹
$ cd guess_num
$ cat README.md
📖 查看项目说明...
3
⚙️

配置项目环境

使用uv sync命令自动配置项目运行所需的环境

🖥️ 环境配置
$ uv sync
📦 正在安装依赖包...
✅ 环境配置完成！
💡
小贴士：uv是一个现代化的Python包管理工具，会自动帮你安装项目需要的所有依赖包！
4
🚀

运行项目

一切准备就绪，现在来运行这个猜数字小游戏！

🖥️ 运行游戏
$ uv run guess_num.py
🎮 猜数字游戏启动中...
🎯 请猜一个1到100之间的数字：
5
🔄

更新项目仓库

项目有新内容？用 git pull 命令一键同步最新代码！

🖥️ 更新项目
$ git pull
🔄 正在更新项目...
✅ 更新完成！
🎉

恭喜！你已经学会了完整的项目使用流程

1 git clone 下载项目
2 阅读 README.md
3 uv sync 配置环境
4 uv run 运行项目
5 git pull 更新项目

现在你可以用同样的方法去探索更多有趣的GitHub项目了！
